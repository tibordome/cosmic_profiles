Classes for Cosmic Profiling
*****************************

The ``profile_classes`` module defines the ``DensProfs``, ``DensShapeProfs``, ``DensProfsHDF5`` and ``DensShapeProfsHDF5`` classes that constitute the heart of the *CosmicProfiles* package, providing methods to perform shape and density profile estimation and fitting.
